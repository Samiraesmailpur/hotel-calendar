# hotel-calendar 
https://hotel-calendar.netlify.app/calendar

## Project setup
```
git clone https://github.com/Samiraesmailpur/hotel-calendar.git
cd hotel-calendar
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
